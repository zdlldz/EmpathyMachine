# Global Drag & Drop Ingestion

## Overview
The Media Library now supports a seamless "Drop Anywhere" ingestion workflow. Instead of navigating through file pickers, you can simply drag your media files from your operating system (Finder, Explorer, etc.) and drop them directly onto the application window to begin the ingestion process.

## Getting Started
1. Open the **Media Library** view.
2. Select one or more image or video files from your computer.
3. Drag the files over the application window. You will see a "Drop to Ingest" overlay appear.
4. Release the files. The system will automatically begin importing them into your library.

## Key Capabilities
- **Frictionless Ingestion:** No need to click "Import" or browse folders; just drag and drop.
- **Batch Processing:** Drop dozens of files at once to queue them for ingestion.
- **Visual Feedback:** A clear, high-contrast overlay provides immediate confirmation that the system is ready to receive your files.
- **Background Operations:** Ingestion happens in the background, allowing you to continue browsing your library.

## Frequently Asked Questions
- **Q: Can I drop folders?**
- **A:** Currently, the system supports dropping individual files or multiple selected files. Folder drop support is planned for a future update.
- **Q: Does this work from my web browser?**
- **A:** Dragging files from a web browser into the application is not yet supported; please use files stored locally on your machine.
