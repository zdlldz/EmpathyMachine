# Organizing Your Media

## Overview
The Media Library now features a professional suite of organization tools. You can now categorize assets using Virtual Collections and Tags, and evaluate them with a 5-star rating system.

## 1. Virtual Collections (!)
Collections are the primary way to group related assets across different folders.
- **Shorthand:** Use `!` when searching or adding.
- **Behavior:** An asset can live in multiple collections without being moved on your hard drive.

## 2. Tags (#)
Tags provide a flexible, flat taxonomy for detailed categorization (e.g., `#nature`, `#client-work`).
- **Shorthand:** Use `#` for quick entry.
- **Batch Tagging:** Select multiple items in the grid to add tags to all of them at once in the Inspector.

## 3. 5-Star Ratings
Evaluate your media using the new rating system. 
- Open the **Item Details** panel (Inspector).
- Click the stars to set a rating from 1 to 5.
- Ratings are stored as metadata and will be used for future "Smart Collection" filtering.

## 4. The Three-Column Layout
Maximize your workspace by toggling the sidebars:
- **Left Rail:** Switch between your **File Tree** and **Filters** (Collections/Tags).
- **Right Rail:** Open the **Item Details** (Inspector) to view technical metadata, ratings, and taxonomies.

## FAQ
- **Q: Can I filter by multiple tags?**
- **A:** Yes! Selecting multiple tags in the Filter rail will show assets that match any of the selected tags.
- **Q: Are sidebars open by default?**
- **A:** No, sidebars are collapsed by default to provide a clean, focused grid view.
