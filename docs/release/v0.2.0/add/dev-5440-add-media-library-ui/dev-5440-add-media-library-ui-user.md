# Your Media Library

The Media Library is the visual heart of your workspace. It allows you to browse, search, and manage thousands of images and videos with blazing-fast performance.

## Importing Media

1.  Click the **Import** button (folder icon) in the top right.
2.  Select a folder from your computer.
3.  **Recursive Scan:** Check this box if you want to import files inside subfolders.
    *   **Depth 0:** Scans everything (Infinite).
    *   **Depth 1:** Scans only the immediate subfolders.
4.  **Managed vs. External:**
    *   **Managed (Default):** Copies files into the app for portability.
    *   **External:** References files where they are (saves disk space).

## Thumbnails & Previews
The app automatically generates high-quality WebP thumbnails for every video and image you import. This happens in the background, so you can keep working while your library populates.

## Supported Formats
We currently support:
- **Images:** JPG, PNG, GIF, WebP
- **Video:** MP4, MOV, WebM
