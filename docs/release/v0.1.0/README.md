# Release Notes: v0.1.0

## Summary
v0.1.0 established the foundation of the project, focusing on the Tauri v2 integration, basic styling, and initial domain models.

## Key Milestones
- **Bootstrap:** Initialized the Rust/Svelte project structure.
- **UI Foundation:** Established the SCSS design system and Bits UI wrappers.
- **Titlebar Refresh:** Implemented edge-to-edge titlebars and custom traffic light insets.
- **Core Features:** Initial implementations of window presets, connection handling, and the prompt engine.
