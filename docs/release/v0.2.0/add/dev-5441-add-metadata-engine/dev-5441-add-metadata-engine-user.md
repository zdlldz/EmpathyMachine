# File Properties

The system automatically extracts technical details for every video and image you import.

## Available Data
We currently track:
- **Resolution:** e.g., "1920x1080"
- **Duration:** Length in seconds.
- **Codec:** e.g., "h264", "hevc".

## How to View
(Future UI Feature) Select a file in the Media Library and open the **Inspector** panel to see these details.
